## Overview

BB Sonic Suite combines multiple web audio utilities into one unified browser workspace. It is aimed at people who need to capture audio, work with samples, shape waveforms, and export finished material without adding desktop software or browser extensions.

The suite is centered on everyday audio workflows that stay local to the browser. You can record from a microphone, load files, manage playback, and export results, which makes it a practical option for fast creation, basic editing, and sound exploration on any system with a current browser.

## Using the Suite

After the app opens, you can work through the tools according to the task at hand:

2. Load an audio file for editing or sampling.
3. Use the tape-style recorder for speed-based playback and punch-in capture.

5. Edit stereo waveforms with the built-in audio editor.
6. Export the finished audio as WAV when you are ready to save the result.
